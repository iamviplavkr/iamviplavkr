<summary><b>🔍 ERP Anomaly Detector — Financial Fraud Detection</b></summary>
<br/>

An ML pipeline combining Random Forest and Isolation Forest models to flag anomalous financial transactions inside ERP data, served through FastAPI with a Streamlit front end.

| Attribute | Detail |
|---|---|
| **Stack** | Python, FastAPI, Streamlit, scikit-learn |
| **Scale** | Batch + on-demand anomaly scoring on ERP transaction records |
| **Performance** | 96% precision, PR-AUC 0.88 |
| **Security** | Isolated inference service via FastAPI |
| **Impact** | Surfaces likely fraudulent transactions for analyst review |
| **Repository** | [github.com/iamviplavkr/erp-anomaly-detector](https://github.com/iamviplavkr) |

This project pairs a supervised classifier with an unsupervised anomaly detector to catch both known fraud patterns and novel outliers, striking a balance between precision and recall that keeps false positives manageable for reviewers.

</details>
